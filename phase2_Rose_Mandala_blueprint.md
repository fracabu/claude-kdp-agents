```

```

           = 8.5" + 0.234" + 8.5" + 0.25" (0.125" x 2)
           = 17.484"

Full Height = Trim Height + Bleed

            = 8.75"

## 4.2 Design Placement Guidelines

### Centering Specifications
- All designs MUST be perfectly centered on page
- Design should not exceed 7.5" x 7.5" safe area
- Maintain minimum 0.5" clear margin from design edge to trim edge
- No design elements should extend into bleed area

### Resolution Requirements
- Minimum 300 DPI at final print size
- Recommended working resolution: 2550 x 2550 pixels (8.5" x 300 DPI)
- Export with bleed: 2625 x 2625 pixels (8.75" x 300 DPI)

## 4.3 Difficulty Progression Order

**RECOMMENDED: Progressive Difficulty (Implemented)**

The designs are ordered Easy (1-18) to Medium (19-40) to Hard (41-52) because:
1. Builds colorist confidence gradually
2. Allows skill development throughout book
3. Creates satisfying sense of accomplishment
4. Industry standard for coloring books
