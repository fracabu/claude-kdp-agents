# 3. Interior Layout Specifications

## 3.1 Page Dimensions

| Measurement | Value | Notes |
|-------------|-------|-------|
| **Trim Size** | 8.5" x 8.5" | Final cut size |
| **Bleed Area** | 0.125" all sides | Images extending to edge must include bleed |
| **Document Size with Bleed** | 8.75" x 8.75" | Full document including bleed |
| **Safe Zone (Inner Margin)** | 0.25" from trim | Keep critical elements inside this |
| **Gutter (Inside Margin)** | 0.375" minimum | Accounts for binding |

## 3.2 Image Placement Guidelines

```
+------------------------------------------+
|          0.125" BLEED ZONE               |
|  +------------------------------------+  |
|  |       0.25" SAFE MARGIN            |  |
|  |  +------------------------------+  |  |
|  |  |                              |  |  |
|  |  |                              |  |  |
|  |  |     COLORING IMAGE AREA      |  |  |
|  |  |        7.75" x 7.75"         |  |  |
|  |  |     (centered on page)       |  |  |
|  |  |                              |  |  |
|  |  |                              |  |  |
|  |  +------------------------------+  |  |
|  |                                    |  |
|  +------------------------------------+  |
|                                          |
+------------------------------------------+
```

**Image Specifications:**
- **Recommended Image Size:** 7.75" x 7.75" (centered with margins)
- **Maximum Image Size:** 8.75" x 8.75" (full bleed, edge-to-edge)
- **Resolution:** 300 DPI minimum (2325 x 2325 pixels at 7.75")
- **Color Mode:** Grayscale or Black & White
- **Line Weight:** 1-3 pt for main outlines, 0.5-1 pt for details
- **Format:** PNG with transparent background or high-quality TIFF

## 3.3 Page Structure

```
INTERIOR STRUCTURE (104 pages total):

Page 1: Title Page (optional, or first coloring page)
Pages 2-103: 51 coloring pages with blank backs
   - Odd pages (right-hand): Coloring illustrations
   - Even pages (left-hand): Blank (prevents bleed-through)
Page 104: Final page (coloring page or back matter)

Alternative with Section Dividers:
- Page 1: Title
- Pages 2-3: Italian Architecture section header / first design
- Pages 4-23: Italian designs (10 pages with blanks)
- Pages 24-25: French section header / first design
... and so on
```

## 3.4 Optional Page Numbers

- **Position:** Bottom center, 0.375" from trim edge
- **Font:** Simple sans-serif, 8-10 pt
- **Style:** Numbers only, no decorative elements
- **Note:** Page numbers are optional for coloring books; many successful books omit them

## 3.5 Section Dividers (Optional)

If using regional section dividers:
- Simple decorative header with region name
- Small iconic illustration (simplified landmark silhouette)
- Light gray or 20% black to avoid distraction
- Can include 3-4 word description
